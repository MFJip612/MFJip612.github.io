# 配置交换机支持 Telnet

## 1.【任务情景】

某学校的网络管理员在设备机房对交换机进行了初次配置后，他希望以后在办公室或出差时也可以对设备进行远程管理。本任务要在交换机上进行适当配置，使其可以实现这一需求。

## 2.【任务分析】

本实验以`Huawei S3700-26C-HI`交换机为例。创建`2`台华为`S3700`交换机，其中`1`台交换机充当终端 PC，使用`Copper Line`连接。假设交换机的 IP 地址为`192.168.1.1`，子网掩码为`255.255.255.0`；计算机的 IP 地址为`192.168.1.2`，子网掩码为`255.255.255.0`，然后进行相应配置。

## 3.【计划制定】

| 序号 | 工作内容             | 所需时间  |
| :--: | -------------------- | --------- |
|  1   | 绘制拓扑结构         | 约`5`分钟 |
|  2   | 配置 PC 终端 IP 地址 | 约`3`分钟 |
|  3   | 配置交换机           | 约`5`分钟 |
|  4   | 验证测试             |           |

## 4.【实验步骤】

| 序号 | 操作内容                   | 步骤                                                                                                                                                                                                                                                                           |
| :--: | -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
|  1   | 搭建网络拓扑结构           | 打开 `eNSP` 并新建拓扑，拖入两台 `Huawei S3700-26C-HI`，其中 `1` 台交换机充当终端 `PC`，使用 `Copper Line` 连接。![](https://vip.123pan.cn/1812581465/ymjew503t0l000d7w32xl9tyginw35ikDIYPDqJ1DIFzAGxxDIrP.png)                                                                |
|  2   | 配置终端 `PC` 的 `IP` 地址 | 配置 `PC` 的 `IP` 地址及其子网掩码                                                                                                                                                                                                                                             |
|  3   | 配置交换机                 | 使用 `PC` 连接到交换机，设置语言模式为中文，进入系统模式。将交换机名称设置为“`SWA`”后，关闭信息中心功能。管理虚拟终端接口，将认证模式设置为密码模式，密码：“`huawei`”。配置用户级别为“`2`”，设定超时时间为 `15` 秒。配置交换机的 `IP` 为“`192.168.1.1/24`”。检查配置是否正确。 |
|  4   | 验证测试                   | 从终端`PC`通过`Telnet`连接至交换机。                                                                                                                                                                                                                                           |
